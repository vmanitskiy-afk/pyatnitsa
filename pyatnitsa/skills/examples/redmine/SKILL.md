# Redmine (EasyRedmine)

Навык для работы с системой управления проектами EasyRedmine.

## Возможности

### redmine.create_project
Создаёт новый проект из шаблона trade_v2.
Параметры:
- name (обязательный): Название проекта
- manager: Менеджер проекта (ФИО или часть фамилии)
- rp: Руководитель проекта (ФИО или часть фамилии)
- ap: Ответственный по проекту

### redmine.my_tasks
Показывает задачи текущего пользователя.
Параметры:
- status: Фильтр по статусу (open/closed/all). По умолчанию: open
- limit: Количество задач. По умолчанию: 10

### redmine.project_status
Краткий отчёт по проекту.
Параметры:
- project: Название или ID проекта

### redmine.create_task
Создаёт задачу в проекте.
Параметры:
- project: Название или ID проекта
- subject: Тема задачи
- description: Описание
- assigned_to: Назначить на (ФИО)
- priority: Приоритет (low/normal/high/urgent)

## Примеры использования

Пользователь: "Создай проект Газпром-Поставка, менеджер Ашкеев, РП Иванов"
→ redmine.create_project(name="Газпром-Поставка", manager="Ашкеев", rp="Иванов")

Пользователь: "Что у меня по задачам?"
→ redmine.my_tasks(status="open")

Пользователь: "Статус по проекту Лукойл?"
→ redmine.project_status(project="Лукойл")
