# Навык files — работа с файлами и папками

Навык предоставляет доступ к рабочей папке пользователя (workspace).
Все пути относительны к workspace. Выход за пределы запрещён.

## Возможности

### Навигация
- `files.list` — список файлов/папок (с размером, датой, типом)
- `files.tree` — дерево структуры папки
- `files.stats` — сводка по папке (типы файлов, размер, последние изменения)

### Чтение
- `files.read` — текстовый файл (txt, csv, json, xml, md, py, js, и др.)
- `files.read_excel` — таблица из xlsx/xls (с выбором листа)
- `files.read_pdf` — текст из PDF
- `files.read_docx` — текст из Word-документа
- `files.read_image` — изображение для анализа (возвращает base64)

### Поиск
- `files.search` — полнотекстовый поиск по содержимому файлов (grep)

### Создание
- `files.write` — создать/перезаписать текстовый файл
- `files.write_csv` — создать CSV из данных
- `files.mkdir` — создать директорию
- `files.copy` — копировать файл
- `files.move` — переместить/переименовать файл

## Использование

Агент должен сначала осмотреть папку через `files.list` или `files.tree`,
затем читать нужные файлы, анализировать и давать ответ.

Для аналитических задач: прочитать данные → обработать → ответить текстом
или создать отчёт через `files.write`.

## Ограничения
- Все пути относительны workspace, `..` запрещён
- Макс. размер для чтения: 10МБ
- Excel: макс. 10000 строк за раз
- Поиск: макс. 100 совпадений
